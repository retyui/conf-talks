Кроссплатформенность

  • Xamarin (Использует Mono, нужны знания С#)
  • (уже в релиз версия, нужны знания Dart)
  • (работает в связке с Angular & Vue)
  • React Native (стремительно развивается, JavaScript)

плюсы +

  • Повторное использование кода между платформами iOS/Android/Web
  • Нет нужды в командах нативных разработчиков
  • Кросплатформенность и скорость разработки
  • Низкий порог вхождения
  • Реальные нативные приложения (никаких web-view)
  • Уровень развития комьюнити
